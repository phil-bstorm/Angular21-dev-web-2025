Avant de commencer cet exercice

Lance l'API json-server qui se trouve dans ce projet avec
npm run fakeApi:secured

Système de login

Fichier de travail: src/app/features/exercise/pages/exercise-http-interceptor/components/authentication

Créer une page de login qui permet de s'enregistrer et de se connecter à l'application.

🔎 Aller voir la documentation de @ovyn/api-server pour trouver quelle route appeler pour s'enregistrer et se connecter!

Une fois connecté vérifier si votre token interceptor fonctionne en récupérant les informations de votre propre user!
