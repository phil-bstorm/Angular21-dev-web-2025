Nous allons créer notre propre pipe pour transformer un nombre Celsius en Fahrenheit. Pour cela, nous allons générer un pipe avec Angular CLI et le personnaliser.

Le dossier de travail étant : src/app/features/demo/pages/demo-pipe/demo-custom-pipe/, je vais devoir préciser ce chemin lors de la création de la pipe

  • Générer un pipe : Utilisez Angular CLI pour générer un pipe.
    ng generate pipe /features/demo/pages/demo-pipe/pipes/CelsiusToFahrenheit
